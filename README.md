# Online Quiz Maker

This is a simple Online Quiz Maker web application built using HTML, CSS, and JavaScript.

## Features
- User login
- Create quizzes with multiple-choice questions
- Attempt quizzes and get instant results
- Delete quiz functionality
- Responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Storage
- Browser localStorage is used to store quiz data.

## How to Run
1. Download the project files
2. Open `index.html` in a browser
